# Aplikasi Pelacakan Kendaraan Pengangkut Sampah 

Jakstrada adalah aplikasi pelacakan (tracking) dari kendaraan pengangkut sampah. Setiap kendaraan pengangkut memiliki titik angkut nya masing â€“ masing, dimana perjalanan kendaraan ini bisa dipantau lewat aplikasi ini. Aplikasi ini memiliki dua jenis pengguna akun, yaitu admin dan driver dimana admin memiliki tugas untuk menginput data driver, data TPS (Tempat Pembuangan Sampah) dan data rute perjalanan kendaraannya. Dibawah ini merupakan tampilannya.
